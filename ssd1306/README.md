# Wiring
## Pi Zero
With the GPIO pins on your right, top left is pin 1, with pin 3 below, and so on. Top right is pin 2 with the pin below being pin 4, and so on.

Pin 3: SDA
Pin 5: SCL
Pin 9: GND
Pin 2: VCC (probably also works with Pin 1 on 3.3V)
